pgAdmin II is a comprehensive PostgreSQL database design and management system for Windows 95/98/ME/NT/2000. It is freely available under the terms of the pgAdmin Public Licence and may be redistributed provided the terms of the licence are adhered to.

pgAdmin II uses the PostgreSQL ODBC Driver (psqlODBC) to communicate with a PostgreSQL server. The software allows the entire server to be managed by using 'DSN-less' connections that are made and managed by the pgSchema object library. All PostgreSQL object types are supported and may be created, dropped and edited to the extent supported by the database.

This software was written as a successor to the original pgAdmin product, which though popular, had limitations in the design that prevented it being taken to the 'next level'. The core of the software (the pgSchema library) has a fully object-oriented design, and manages all database operations for the main program, as well as providing a hierarchical object model to represent the existing objects on the server. In order to run install and run pgAdmin II, you will need the following software pre-installed on your PC:

  • The Windows Installer. This is pre-installed on Windows 2000 and ME, but may be required on earlier versions of Windows. Unfortunately Microsoft re-organise their website regularly so you will need to search for 'windows installer redistributable' in order to find the actual downloadable file.
     
  • Microsoft Data Access Components (MDAC). Unlike it's predecessor, pgAdmin II is not dependent on the latest version of MDAC requiring version 2.0 or higher. MDAC is pre-installed on most newer versions of Windows.

Note that the PostgreSQL ODBC driver is not required, as the latest version ships and installs with pgAdmin II.

Mailing Lists

pgAdmin is supported, developed and discussed through mailing lists. Currently there are 2 lists:

  • pgadmin-support@postgresql.org

    This is the primary support list intended for end user support & bug reports. To subscribe to this list, send an email to pgadmin-support-request@postgresql.org with the word 'subscribe' (without the quotes) as the body of message.

    You can search the support list archives here.
     
  • pgadmin-hackers@postgresql.org

    This is the pgAdmin developers list and is intended for discussions on the more technical side (i.e. how pgAdmin does and should work, and new code that's being developed). You are more than welcome to subscribe and contribute to or start threads, but please remember to use pgadmin-support@postgresql.org for end user support issues. To subscribe to this list, send an email to pgadmin-hackers-request@postgresql.org with the word 'subscribe' (without the quotes) as the body of message.

    You can search the hackers list archives here.

To unsubscribe from a list, follow the procedure for subscribing but use 'unsubscribe' in the message body instead of 'subscribe'.

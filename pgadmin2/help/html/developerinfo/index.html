Developer Info

pgAdmin II is an open source application written using Microsoft's Visual Basic 6.0 Service Pack 5. It is built around 4 main components:

  • pgSchema is an object library that provides a comprehensive object model representing an entire PostgreSQL server. pgSchema is responsible for all database operations and logging.

    •  

  • pgAdmin is the core host application. It provides the main user interface for database object browsing and management and allows the execution of arbitrary SQL statements to an editable grid or Exporter.

    •  

  • Exporters are ActiveX DLLs that are passed an ADO recordset by pgAdmin and export it to a file, program or database.

    •  

  • Plugins are ActiveX DLLs that provide extra functionality not provided by pgAdmin itself. Plugin access is provided via a dynamic menu.

pgAdmin II uses a version numbering scheme similar to the Linux Kernel. An odd minor version number indicates a 'development' version. These versions are largely untested and may be unstable. An even minor version number indicates a release quality build.

The source code for pgAdmin is available from our CVS repository and is written to these guidelines.

If you fancy lending a hand, check out the Known Bugs and To Do lists. Before you start though, it's a good idea to let us know on the pgadmin-hackers@postgresql.org mailing list just in case one of us has already started or looked at the item you're interested in.

Download pgAdmin II

Before downloading pgAdmin II or any of it's components, you may want to check that you have all the pre-requisite software:

  • PostgreSQL 7.1 or higher.
     
  • Microsoft Windows 95, 98, ME, NT4 or 2000. Windows XP should work but is currently untested.
     
  • Microsoft Windows Installer.
     
  • Microsoft MDAC 2.0 or higher.

The following files are available for download at present:
